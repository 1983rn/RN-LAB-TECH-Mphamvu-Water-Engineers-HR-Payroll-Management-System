## TROUBLESHOOTING

### Problem: "No module named 'flask'"
**Solution:** Run INSTALL_DEPENDENCIES.bat with internet connection

### Problem: "Port 5001 is already in use"
**Solution:** 
- Close any other programs using port 5001
- Or restart your computer

### Problem: "Connection error" during installation
**Solution:**
- Check your internet connection
- Try again later
- Or use mobile hotspot

### Problem: Application starts but browser shows error
**Solution:**
- Wait 10 seconds after starting
- Try: http://127.0.0.1:5001
- Clear browser cache
- Try different browser (Chrome, Firefox, Edge)
